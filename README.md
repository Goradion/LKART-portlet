-> Inhalte 
-> Dokumente & Medien
-> oben rechts Menü-Icon anklicken (drei Punkte) 
-> Berechtigungen des Basisordners 
-> Option "Anzeigen" für bestimmte Rollen abwählen
-> Speichern
